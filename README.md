# Agent Mobile Money

## Build APK via GitHub Actions

1. Crée un nouveau repo sur GitHub
2. Upload TOUS les fichiers de ce ZIP (pas le ZIP lui-même)
3. Va dans Actions → Build APK → Run workflow
4. Attends 3-4 minutes
5. Télécharge l'APK dans Artifacts

## Installation
- Désinstalle l'ancienne version si tu as un conflit
- Ouvre l'APK → Installer
- Accepte les permissions SMS au premier lancement
